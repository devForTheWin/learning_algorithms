implode — Объединяет элементы массива в строку

explode — Разбивает строку с помощью разделителя.
Возвращает массив строк, полученных разбиением
строки string с использованием separator в качестве разделителя.

isset - определяет, установлена ли переменная.

strlen — Возвращает длину строки

str_repeat — Возвращает повторяющуюся строку
str_repeat(string $string, int $times): string
$string -повторяемая строка. $times - количество повторов (больше или равен нулю, если 0, то возврат пустая строка)
